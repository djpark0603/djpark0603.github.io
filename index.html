Biography

소개

이 사이트는 GitHub Pages로 운영하는 개인 홈페이지입니다. 화려한 랜딩 페이지보다는, 현재 어떤 작업을 하고 있는지 빠르게 파악할 수 있는 구조를 목표로 삼았습니다.

상단에는 기본 프로필과 링크를 배치하고, 본문에는 관심 분야와 최근 프로젝트, 업데이트 메모를 순서대로 정리했습니다. 지금은 기본 버전이지만, 추후 실제 증명사진과 자세한 이력, 프로젝트 설명을 추가할 수 있게 구성해 두었습니다.
